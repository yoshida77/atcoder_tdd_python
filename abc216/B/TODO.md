- [ ] Si = Sj かつ Ti = Tjを満たす整数対(i,j)が存在するかどうか判定する
- [ ]

# 制約
2≤N≤1000
N は整数
S_i,T_iは英小文字のみからなる長さ1以上10以下の文字列

# 入力
N:行数
S_i, T_i

# 出力
同姓同名であるような人の組が存在するならYesを、存在しないならNoを出力せよ

# 例
##
3
tanaka taro
sato hanako
tanaka taro

Yes
##
3
saito ichiro
saito jiro
saito saburo

No
##
4
sypdgidop bkseq
bajsqz hh
ozjekw mcybmtt
qfeysvw dbo

No
