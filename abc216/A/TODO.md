# 問題文
実数 X.Y が与えられます。ただし Y はちょうど 1 桁です。

0≤Y≤2 ならば、X-
3≤Y≤6 ならば、X
7≤Y≤9 ならば、X+
と出力してください。

## 制約
1≤X≤15
0≤Y≤9
X と Y は整数
## 入力
入力は以下の形式で標準入力から与えられる。
```shell
X.Y
```
## 出力
答えを出力せよ。

## ex.1
15.8 -> 15+
## ex.2
1.0 -> 1-
## ex.3
12.5 -> 12
